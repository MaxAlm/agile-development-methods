Klienttekniker

HTML

HTML är en förkortning av Hyper Text Markup Language och är egentligen inte ett programmeringsspråk utan en samling markeringskoder. HTML är en av webbens grundläggande byggstenar tillsammans med CSS (Cascading Style Sheets) och JavaScript. Anledningen till att HTML inte är ett programmeringsspråk är för att HTML inte kan ta emot kommandon, något som krävs i ett programmeringsspråk. Kod i HTML har endast i uppgift att ge struktur.

CSS

CSS är en förkortning av Cascading Style Sheets och är en av webbens tre grundläggande byggstenar. När man bygger en hemsida kan man säga att CSS är webbens ansikte utåt och sköter allt som har med design att göra. Med CSS kan man åstadkomma allt från att välja typsnitt på text till att göra snygga animationer. Precis som HTML så räknas CSS inte som ett programmeringsspråk, då det saknar möjligheten att utföra riktiga kommandon.

JavaScript

När man bygger en hemsida så står HTML för strukturen, CSS för designen och JavaScript för funktionaliteten. Enkla animationer kan idag åstadkommas med hjälp av CSS, men generellt så är det JavaScript som möjliggör allt som händer på en webbplats. Om du har ett formulär på din hemsida, så kan man med hjälp av JavaScript bland annat kolla om användaren har skrivit in fel uppgifter, eller om något inmatnings fält är tomt.
